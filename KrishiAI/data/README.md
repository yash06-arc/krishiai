## KrishiAI datasets

- `prices_karnataka.csv`: Auto-generated on first backend run (or run `python data_gen.py` inside `backend/`).
- Schema:
  - `Crop`
  - `District`
  - `Date` (YYYY-MM-DD)
  - `Price` (₹/kg)

This demo dataset covers **29 districts of Karnataka** and **20 major Indian crops** for investor pitching.
